# Source Registry

## Automatic sources

These sources can be polled automatically right now:

- Kyiv city official portal
- Suspilne Kyiv RSS
- Vechirniy Kyiv RSS
- Big Kyiv RSS
- Kyiv Regional Administration RSS
- Ukrinform RSS
- Ukrainska Pravda RSS

## Manual / curated sources

These are valuable but need manual review, API access, or separate parser work:

- Facebook groups
- Telegram channels
- Instagram pages
- Google Maps
- Waze
- OLX
- local resident chats
- DSNS and National Police sites without direct RSS integration

## Rule

If one automatic source does not respond, the bot skips it and continues with the next one.
