# Publishing Rules

## News

- Publish only if the item matches Chaika / Bucha / Sofiivka / Kyiv region / transport / utilities / safety.
- If the item is about Odessa, general cyber security, or unrelated national topics, skip it.

## Offers

- Publish as a short text digest.
- Keep it brief and useful.

## Coffee of the day

- Publish only if a cafe spot has an image.
- If there is no image, skip the coffee post.

## App link

- Always append the app download link at the end of Telegram posts.
